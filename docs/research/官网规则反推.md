# 官网规则反推

调研对象：https://www.cvdrisk.com.cn/ASCVD/Eval/

调研日期：2026-05-08

## 1. 官网提交方式

官网前端不直接计算风险值。

表单提交流程：

1. 点击提交按钮。
2. 前端将表单序列化，调用 `POST /ASCVD/Eval/ValidInput` 做后端校验。
3. 校验通过后，表单提交到 `POST /ASCVD/Eval/Result`。
4. 后端返回已经渲染好的结果页，包括：
   - 10 年风险值
   - 理想危险因素状态下 10 年风险值
   - 终生风险值
   - 理想危险因素状态下终生风险值
   - 风险等级
   - 温馨建议

## 2. 问卷字段

官网表单字段如下：

| 字段 | 参数名 | 取值 |
| --- | --- | --- |
| 性别 | `sex` | 男 `1`，女 `2` |
| 年龄 | `age` | 20-85 |
| 南北方 | `region` | 南方 `0`，北方 `1` |
| 城乡 | `area` | 农村 `0`，城市 `1` |
| 腰围 cm | `waist` | 50-130 |
| 总胆固醇单位 | `tc_unit` | mg/dl `1`，mmol/L `2` |
| 总胆固醇 | `tc` | mg/dl: 80-400；mmol/L: 2-11 |
| HDL-C 单位 | `hdlc_unit` | mg/dl `1`，mmol/L `2` |
| HDL-C | `hdlc` | mg/dl: 20-130；mmol/L: 0.5-4 |
| 收缩压 | `sbp` | 70-200 |
| 舒张压 | `dbp` | 40-140 |
| 服用降压药 | `drug` | 是 `1`，否 `0` |
| 糖尿病 | `dm` | 是 `1`，否 `0` |
| 当前吸烟 | `csmoke` | 是 `1`，否 `0` |
| 心脑血管病家族史 | `fh_ascvd` | 是 `1`，否 `0` |

血脂单位换算：

```text
mg/dl = mmol/L / 0.0259
```

## 3. 10 年风险工程近似公式

公开论文中的 10 年风险公式为 Cox 模型。官网后端应使用了更高精度系数；论文 PDF 公开表格中的系数有四舍五入，直接计算会和官网结果有小偏差。

为复刻官网输出，当前按官网黑盒输出拟合出如下工程近似模型：

```text
z = Σ(coeff * feature) + intercept
10_year_risk = 1 - exp(-exp(z))
```

输出为小数，例如 `0.047` 表示 `4.7%`。

连续变量使用自然对数：

```text
ln_age = ln(age)
ln_sbp = ln(sbp)
ln_tc = ln(tc_mg_dl)
ln_hdlc = ln(hdlc_mg_dl)
ln_waist = ln(waist)
```

分类变量：

```text
smoke: 当前吸烟，是 1，否 0
dm: 糖尿病，是 1，否 0
north: 北方 1，南方 0
urban: 城市 1，农村 0
family: 家族史，是 1，否 0
```

| `ln_age_x_family` | -1.536263 |
| `intercept` | -145.726914 |

男性计算说明：

```text
如果 drug = 1：
  ln_treated_sbp = ln(sbp)
  ln_untreated_sbp = 0
  ln_age_x_ln_treated_sbp = ln(age) * ln(sbp)
  ln_age_x_ln_untreated_sbp = 0

如果 drug = 0：
  ln_treated_sbp = 0
  ln_untreated_sbp = ln(sbp)
  ln_age_x_ln_treated_sbp = 0
  ln_age_x_ln_untreated_sbp = ln(age) * ln(sbp)

ln_age_x_smoke = ln(age) * smoke
ln_age_x_family = ln(age) * family
```

## 4. 终生风险反推进展

终生风险的原始精确公式仍未找到完整公开系数。公开论文说明其模型为 Fine-Gray 竞争风险模型，计算从当前年龄到 85 岁的累计 ASCVD 风险；但公开补充材料未包含可直接编码的完整系数表。

当前已完成两类黑盒拟合：

### 4.1 线性 logit 近似模型

使用 360 条官网结果样本，其中男性 180 条、女性 180 条。年龄范围限定为 `20-59`，因为官网只在 60 岁以下展示终生风险。

模型形式：

```text
logit(lifetime_risk) = Σ(coeff * feature)
lifetime_risk = 1 / (1 + exp(-z))
```

特征包括：

```text
intercept
age
age^2
ln(age)
ln(tc)
ln(hdlc)
ln(waist)
ln(treated_sbp)
ln(untreated_sbp)
smoke
dm
north
urban
family
age 与主要风险因子的交互项
```

误差：

```text
男性：平均绝对误差约 1.81 个百分点，P95 约 4.86 个百分点
女性：平均绝对误差约 1.84 个百分点，P95 约 4.57 个百分点
```

这个模型只适合做趋势判断，不建议作为 demo 的默认实现。

### 4.2 二阶多项式 ridge 近似模型

为了提高工程复刻精度，使用标准化基础特征及其二阶交叉项拟合：

```text
base_features:
age, age^2, ln(age), ln(sbp), ln(tc), ln(hdlc), ln(waist),
drug, smoke, dm, north, urban, family,
drug * ln(sbp), (1 - drug) * ln(sbp)

poly_features:
1 + normalized(base_features) + normalized(base_features) 的二阶交叉项
```

模型文件：

```text
docs/research/lifetime_poly2_model.json
```

采样数据：

```text
docs/research/lifetime_fit_samples.jsonl
```

误差：

```text
男性：全量样本 MAE 约 0.60 个百分点，P95 约 1.70 个百分点；留出集 MAE 约 1.23 个百分点
女性：全量样本 MAE 约 0.64 个百分点，P95 约 1.57 个百分点；留出集 MAE 约 1.43 个百分点
```

使用建议：

1. 该模型可以作为 demo 的临时终生风险近似实现。
2. 必须在 UI 或代码注释中标注“工程拟合结果，非原始 China-PAR 终生风险公式”。
3. 如果 demo 用于严肃医学或保险决策场景，应继续寻找原始终生风险系数，或接入权威计算服务。

## 5. 风险分层

### 10 年风险

```text
risk10 < 5%        低危
5% <= risk10 < 10% 中危
risk10 >= 10%      高危
```

### 终生风险

```text
lifetime_risk < 32.8%  低危
lifetime_risk >= 32.8% 高危
```

终生风险只在年龄 `< 60` 时展示。年龄 `60` 及以上，官网结果页不展示终生风险。

## 6. 理想危险因素状态

官网结果页说明的理想危险因素状态：

```text
总胆固醇 < 200 mg/dl，即 < 5.18 mmol/L
HDL-C >= 40 mg/dl，即 >= 1.04 mmol/L
未治疗收缩压 < 120 mmHg
舒张压 < 80 mmHg
男性腰围 < 90 cm
女性腰围 < 85 cm
当前不吸烟
没有糖尿病
无心脑血管病家族史
年龄、性别、现居住地区使用用户输入值
```

## 7. 理想风险生成规则

官网会同时返回“理想危险因素状态下”的 10 年风险和终生风险。

当前已确认：

1. 理想 10 年风险使用同一套 10 年风险公式计算。
2. 理想终生风险使用同一套终生风险后端模型计算。
3. 年龄、性别、南北方、城乡沿用用户输入。
4. 其他危险因素替换为固定阈值状态。
5. 官网文案写的是“应小于”，但后端实际返回的是一个具体百分比。

理想状态替换值：

| 字段 | 替换值 |
| --- | --- |
| `tc` | `200 mg/dl`，即约 `5.18 mmol/L` |
| `hdlc` | `40 mg/dl`，即约 `1.04 mmol/L` |
| `sbp` | `120 mmHg` |
| `dbp` | `80 mmHg` |
| `drug` | `0`，未服用降压药 |
| 男性 `waist` | `90 cm` |
| 女性 `waist` | `85 cm` |
| `dm` | `0` |
| `csmoke` | `0` |
| `fh_ascvd` | `0` |

验证方式：

1. 构造输入值刚好等于上述阈值。
2. 官网返回的实际风险值与理想风险值相同。
3. 构造更优值，例如 `tc=199`、`hdlc=41`、`sbp=119`、`dbp=79`、腰围低于阈值，实际风险会低于理想风险，说明官网不是取“更优状态”，而是取阈值点。

采样文件：

```text
docs/research/ideal_risk_samples.jsonl
```

注意：这与指南文案的严格不等号略有差异。例如指南写 `TC < 200`、`未治疗 SBP < 120`，但官网后端理想风险计算取的是 `TC=200`、`SBP=120`。

## 8. 温馨建议规则

### 8.1 基础建议

基础建议由 10 年风险等级决定。

低危：

```text
应该接受生活方式指导，以保持自身的低风险状况，加强自我监测。
```

中危：

```text
应积极改变不良生活方式，如戒烟、控制体重、增加体力活动等，如有必要可以在临床医生指导下进行相关治疗。
```

高危：

```text
应积极改变不良生活方式，如戒烟、控制体重、增加体力活动等，同时应该针对自身危险因素，在临床医生指导下进行降压、调脂、降糖等药物治疗。至少每年进行一次体检，必要时可以进行心脏超声、颈动脉超声等详细的影像学检查，以进一步评估心脑血管病风险。
```

### 8.2 附加建议

终生风险高危：

```text
触发条件：年龄 < 60，终生风险 >= 32.8%，且 10 年风险不是高危
文案：鉴于您终生风险处于高危水平，请密切注意相关危险因素，采取必要干预措施。
```

中心型肥胖：

```text
触发条件：男性腰围 >= 90 cm，或女性腰围 >= 85 cm
文案：您为中心型肥胖，应积极改变生活方式，控制体重和腰围。
```

未服药高血压：

```text
触发条件：drug = 0，且 sbp >= 140 或 dbp >= 90
文案：您为高血压，应密切关注血压，必要时就医，进行治疗。
```

服药后血压控制不佳：

```text
触发条件：drug = 1，且 sbp >= 140 或 dbp >= 90
文案：您血压控制不佳，应咨询专业医师，进一步采取措施，控制血压。
```

高胆固醇血症：

```text
触发条件：tc >= 240 mg/dl，或 tc >= 6.21 mmol/L
文案：您为高胆固醇血症，应密切关注总胆固醇水平，必要时就医，进行治疗。
```

糖尿病：

```text
触发条件：dm = 1
文案：请积极治疗糖尿病，避免并发症的发生。
```

吸烟：

```text
触发条件：csmoke = 1
文案：吸烟是心脑血管病的重要危险因素，建议立即戒烟。
```

### 8.3 未观察到单独触发建议的因素

以下因素参与风险计算，但当前黑盒测试未观察到单独建议文案：

- HDL-C 偏低
- 心脑血管病家族史
- 南北方
- 城乡

## 10. 当前仍缺什么

### 10.1 终生风险原始公式

仍缺原始 Fine-Gray 模型的完整系数、基线累计风险或等价参数。当前二阶多项式模型是黑盒工程拟合，不是医学文献原始公式。

### 10.2 理想风险具体替换值

已确认，见第 7 节。后续只需要在 demo 代码中固化这些替换值，并用官网样例做回归测试。

### 10.3 后端校验完整规则

已新增系统枚举样本：

```text
docs/research/valid_input_cases.jsonl
```

当前结论：

1. 所有表单字段必填；缺失时返回 `评测项不得为空`。
2. 年龄边界为闭区间 `20-85`，只接受整数。`20.5` 会返回 `评测项不得为空`。
3. 腰围边界为闭区间 `50-130`，允许小数。
4. 总胆固醇：
   - `mg/dl`：闭区间 `80-400`，允许小数。
   - `mmol/L`：闭区间 `2-11`，允许小数。
5. HDL-C：
   - `mg/dl`：闭区间 `20-130`，允许小数。
   - `mmol/L`：闭区间 `0.5-4`，允许小数。
6. 收缩压边界为闭区间 `70-200`，只接受整数。`115.5` 会返回 `评测项不得为空`。
7. 舒张压边界为闭区间 `40-140`，只接受整数。`75.5` 会返回 `评测项不得为空`。
8. `region`、`area`、`tc_unit`、`hdlc_unit`、`drug`、`dm`、`csmoke`、`fh_ascvd` 均有后端枚举校验。
9. `sex=9` 在 `ValidInput` 中未被拦截，但前端只提供 `1/2`。demo 实现必须按前端枚举限制性别为男 `1`、女 `2`。

舒张压说明：

1. 当前 10 年风险公式不使用 `dbp`。
2. 官网结果显示，单独改变 `dbp` 不改变风险值。
3. `dbp` 参与输入校验、理想状态展示和高血压建议触发。

### 10.4 建议规则边界

主要建议规则已反推，但仍需扩大验证：

```text
10 年高危且终生高危时，官网目前未展示终生高危附加建议，需要更多样本确认
HDL-C 低值目前未单独触发建议，需要验证极端低值
家族史目前未单独触发建议，需要验证不同年龄/风险等级组合
```
